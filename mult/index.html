The Developers

Mult was developed by Jackson Martin, and Cole Vahey for a computer programming class at school. All of the code is uploaded to GitHub.

Mult on GitHub

Screenshots

  • The UI

    Mult gives you two options: Scan for new messages? Or send a message? You can select this by typing 1 or 2.

  • Sending a message

    Mult will prompt you to give it a message. That message is logged to a .talk.txt file, and a .history.txt file.

  • Message alert

    Mult will notify you when you get a new message. Once you log in, it will say NEW MESSAGE. After that, the notification goes away.

Mult takes a bit of setup, but it's worth it.

Written in Python and Bash.

My thoughts on Mult.

Mult was written with one thing in mind: "How can I make a simple, yet efficient IM Program?" I had used ytalk before, but I hated that the user had to be online to talk. I wanted something more like the standard messaging app, but on the command line. We took an idea, and made it a product.
